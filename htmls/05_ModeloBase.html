Modelo Base

En este notebook, estableceremos nuestro punto de partida para la modelizacin predictiva en nuestro proyecto. El objetivo principal es construir y evaluar un modelo base utilizando un Dummy Classifier. Este enfoque nos permitir establecer una lnea base para el rendimiento, la cual es crucial para evaluar la eficacia de modelos ms avanzados posteriormente.

Un Dummy Classifier es un tipo de modelo que no aprende nada de los datos. Realiza predicciones utilizando reglas simples y no informadas, como predecir la clase ms frecuente o generar predicciones aleatorias. Aunque suena contraproducente, este modelo juega un papel fundamental en la construccin de sistemas predictivos. Nos ayuda a comprender el mnimo rendimiento que cualquier otro modelo debera superar para ser considerado til.

def evaluate_model(ytest, ypred, ypred_proba = None):
    if ypred_proba is not None:
        print('ROC-AUC score of the model: {}'.format(roc_auc_score(ytest, ypred_proba[:, 1])))
    print('Accuracy of the model: {}\n'.format(accuracy_score(ytest, ypred)))
    print('Classification report: \n{}\n'.format(classification_report(ytest, ypred)))

La interpretacin de las mtricas revela una situacin comn al trabajar con un modelo base como el Dummy Classifier: el rendimiento extremadamente desigual entre las clases. Este modelo est diseado para predecir siempre la clase ms frecuente, y por lo tanto, tiene un alto rendimiento para la clase dominante (clase 0 en este caso) y un rendimiento nulo para la clase minoritaria (clase 1). Es importante recalcar que estas mtricas estn en lnea con lo esperado para un Dummy Classifier y sirven para establecer una lnea de base muy bsica. Cualquier modelo posterior que se desarrolle deber superar significativamente estas mtricas para ser considerado valioso, especialmente en trminos de precisin y recall para la clase minoritaria, la cual es completamente ignorada por el Dummy Classifier. Este enfoque nos ayuda a establecer el mnimo umbral de rendimiento y a ilustrar la necesidad de tcnicas ms sofisticadas para manejar desequilibrios de clases y mejorar la precisin en la prediccin de todas las categoras.
